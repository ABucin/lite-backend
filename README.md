lite-backend
============

Backend for the LITE project. 

### Pre-requisites:

1. Install Node.js from `http://nodejs.org/` .
2. Install MongoDB from `http://www.mongodb.org/downloads` .

### Technology stack

* Node.js
  * express
  * passport
  * mocha
  * nodemon
* MongoDB
